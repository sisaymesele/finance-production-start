How It Works

PayrollPro Workflow

Configure Company Info

Set up your organization information.

Add Employee Information

Create personnel records, employment details, and financial info.

Select Payroll Month & Components

Choose the payroll month and define components like basic salary, overtime, allowances, deductions etc.

Process Payroll & Adjustments
Regular Payroll Processing

Go to Regular Payroll → Manage Payroll by Month → Select month → Add personnel and financials data (process multiple time if employee and month already configured).

Reports, Journals & Analytics for Regular Payroll

Auto-generate journal entries, performance reports, audit logs, and expense summaries.

Earnings and Deduction Adjustments
Earning Adjustments

Select the recording month, then the month to be adjusted.
Use this to apply earning adjustments such as retroactive salary increases, missed allowances or bonuses, back pay due to late promotions or rate changes, and other positive corrections to employee earnings.

Deduction Adjustments

Select the recording month, then the month to be adjusted.
Use this to apply deduction adjustments such as back deductions, correction of overpayments, missed tax or pension deductions, and other retroactive deduction corrections.

Reports, Journals & Analytics for Combined Payroll

Auto-generate journal entries, performance reports, audit logs, and expense summaries for the combined summary report for regular payroll and adjustments.

Severance Workflow

1. Select Date

Choose severance year and month.

2. Severance Type

Select the severance type applicable to the employee.

3. Daily Wage

Enter daily wage or last week's average salary.

4. Service Period

Input service start and end dates.

5. Final Report

Auto-generate severance journal entries, reports, audit logs, and summaries.

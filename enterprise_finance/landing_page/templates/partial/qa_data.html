How Regular Payroll Work?

Enter employee financial information including earnings and deductions. Select the payroll month and relevant components, then go to “Make Payroll by Month” and choose the month to process. Add or update earnings and deductions as needed—multiple personnel can be processed in the same month. The system automatically calculates gross pay, tax, pension contributions, and net pay. Comprehensive reports, journal entries, and expense summaries are auto-generated for transparency and compliance.

How does adjustment work?

Payroll adjustments are used when an error or update is discovered after the original payroll has been processed. This can include missing earnings, incorrect deductions, or additional earnings/deductions that need to be applied.

To process an adjustment:

  1. Select the current payroll: the payroll record where the adjustment will be recorded.
  2. Choose the payroll needing adjustment: the original payroll period that contains the error or requires correction.
  3. Specify the adjustment details: select the payroll component to adjust (e.g., basic salary, allowance, deduction) and provide the reason (e.g., underpayment, overpayment, correction).
  4. Enter the adjustment amount: the system will automatically calculate taxable and non-taxable portions.
  5. Review updates: the system updates the corrected values in the original payroll period (adjusted fields) and updates totals in the current payroll period (recorded fields), including tax, pension, and expenses.

Following this process ensures accurate payroll records and proper reflection of all adjustments across both the original and current payroll periods.

How to Process Unpaid Salary for Multiple Months at Once?

To process unpaid salaries for multiple months:

  1. Step 1: In the Regular Payroll table, set the salary amount to 0 for each month that was unpaid.
  2. Step 2: In the Earning Adjustment table, select each month with 0 salary and enter the correct salary amount individually for each month.
  3. Step 3: The system will automatically calculate the taxable and non-taxable portions and update the payroll totals accordingly.
